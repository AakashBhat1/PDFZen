# Dev / scratch scripts

Ad-hoc helpers used during converter development. Not part of the app runtime.

| Script | Purpose |
|--------|---------|
| `convert_pdf.py` | One-off pdf2docx conversion against a sample PDF |
| `inspect_pdf.py` | Print PyMuPDF text blocks/images for a sample PDF |
| `split_tools.py` | One-shot splitter that broke mega tool files into feature folders (already applied) |

Sample fixtures live in `public/samples/` (gitignored; used by UI “Load Test PDF” buttons and these scripts).
